Pretty-Printing

Documents

A document is a structural collection of text. The following are the various types of documents, usually created with various combinators provided below. 

Empty ⇒ Class

The empty document, with a height and width of 0.

Beside(@left, @right, @space?) ⇒ Class

A document comprised of two documents positioned beside each other, with an optional space in-between.

Above(@above, @below, @overlap?) ⇒ Class

A document comprised of two documents positioned above and below each other, optionally overlapping if possible.

Nest(@body, @depth) ⇒ Class

A document indented to a given depth.

Text(@value) ⇒ Class

A document containing arbitrary text, with a height of 1.

Doc empty? ⇒ Boolean

Trivial emptiness check. true for Empty, false for everything else.

Example:

> doc { empty } empty?
true
> doc { text("abc") } empty?
false
> doc { text("") } empty?
false

Doc width ⇒ Integer

Calculate the width of the document, in characters.

Example:

> doc { empty } width
0
> doc { text("abc") } width
3

Doc height ⇒ Integer

Calculate the height of the document, in lines.

Note that an empty document has a height of 0.

Example:

> doc { empty } height
0
> doc { text("abc") } height
1

Doc render ⇒ String

Render the document as a String.

Example:

> doc { empty } render
""
> doc { text("abc") } render
"abc"
> '(1 + 1) pretty render
"1 + 1"

Constructing

(left: Doc) <+> (right: Doc) ⇒ Doc

Position one document beside another, separated by a space, unless either side is empty.

Example:

> doc { text("x") <+> value(42) }
x 42
> doc { empty <+> value(42) }
42

(left: Doc) <> (right: Doc) ⇒ Doc

Position one document beside another unless either side is empty.

Example:

> doc { text("x") <> value(42) }
x42
> doc { empty <> value(42) }
42

(above: Doc) // (below: Doc) ⇒ Doc

Position one document above another, unless either are empty.

If the last line of the first argument stops at least one position before the first line of the second begins, these two lines are overlapped.

Example:

> doc { text("hi") // text("there") nest(1) }
hi
 there
> doc { text("hi") // text("there") nest(5) }
hi   there

(above: Doc) /+/ (below: Doc) ⇒ Doc

Position one document above another, without overlapping, unless either are empty.

Example:

> doc { text("hi") /+/ text("there") nest(1) }
hi
 there
> doc { text("hi") /+/ text("there") nest(5) }
hi
     there

Doc nest(depth) ⇒ Doc
  | depth is-a?(Integer)

Indent the document to the given depth. For Nest, this just increases its indentation level.

Example:

> doc { text("hi") nest(5) }
     hi
> doc { text("hi") nest(5) nest(6) }
           hi

(delimiter: Doc) punctuate(documents: List) ⇒ List
  | documents all?(&#is-a?(Doc))

Intersperse delimiter document through documents.

Example:

> doc { semi punctuate([value(1), value(2), value(3)]) }
[1;, 2;, 3]
> doc { hsep(semi punctuate([value(1), value(2), value(3)])) }
1; 2; 3

Helpers

doc(&body)

Shortcut for Doc onto(&body). This provides the various helper methods below, which are also available in the Pretty module.

Shortcuts

These shortcuts are primarily for quickly creating Text documents, especially for things commonly found in syntax. 

text(s) ⇒ Doc
  | s is-a?(String)

Create a Text document with the given contents.

value(x) ⇒ Doc

Create a Text document with x inspect.

semi ⇒ Doc

Shortcut for text(";").

comma ⇒ Doc

Shortcut for text(",").

colon ⇒ Doc

Shortcut for text(":").

space ⇒ Doc

Shortcut for text(" ").

equals ⇒ Doc

Shortcut for text("=").

lparen ⇒ Doc

Shortcut for text("(").

rparen ⇒ Doc

Shortcut for text(")").

lbrack ⇒ Doc

Shortcut for text("[").

rbrack ⇒ Doc

Shortcut for text("]").

lbrace ⇒ Doc

Shortcut for text("{").

rbrace ⇒ Doc

Shortcut for text("}").

Wrapping

More shortcuts for common cases in pretty-printing, dealing with wrapping a document in delimiters. 

parens(d) ⇒ Doc

Wrap a document in parentheses.

brackets(d: Doc) ⇒ Doc

Wrap a document in brackets ([]).

braces(d: Doc) ⇒ Doc

Wrap a document in braces ({}).

quotes(d: Doc) ⇒ Doc

Wrap a document in single-quotes.

double-quotes(d: Doc) ⇒ Doc

Wrap a document in double-quotes.

Combining

Helpers for creating a single document from many. 

empty ⇒ Doc

An empty document. The identity for <>, <+>, //, /+/, and anywhere in the following methods.

hcat(ds) ⇒ Doc
  | ds all?(&#is-a?(Doc))

Reduce a list of documents with <>.

hsep(ds) ⇒ Doc
  | ds all?(&#is-a?(Doc))

Reduce a list of documents with <+>.

vcat(ds) ⇒ Doc
  | ds all?(&#is-a?(Doc))

Reduce a list of documents with //.
